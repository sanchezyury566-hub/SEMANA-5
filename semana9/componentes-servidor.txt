Sistema operativo:
Es el software principal del computador que permite ejecutar programas y administrar recursos. Ayuda a que Node.js y el servidor funcionen correctamente.

Node.js:
Es un entorno que permite ejecutar JavaScript fuera del navegador. Se utiliza para crear y ejecutar servidores web como el de esta actividad.

Aplicación server.js:
Es el archivo donde está el código del servidor. Define las rutas y las respuestas que verá el usuario al ingresar al navegador.

Puerto:
Es el canal de comunicación donde el servidor espera solicitudes. En esta práctica el servidor escucha en el puerto 3000.

Logs:
Son registros que muestran eventos y actividades del servidor. Sirven para identificar errores y monitorear el funcionamiento.
